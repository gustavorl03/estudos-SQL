# 🗄 SQL para Análise de Dados

Este repositório contém meus estudos e exercícios práticos utilizando **SQL para análise de dados**.

Aqui estão consultas, exercícios e projetos focados em manipulação, exploração e análise de dados em bancos relacionais.

---

## 🎯 Objetivos de Aprendizado

- Aprender a consultar dados utilizando SQL
- Trabalhar com filtros e agregações
- Combinar dados de múltiplas tabelas
- Resolver problemas reais de análise de dados
- Criar consultas para geração de insights

---
